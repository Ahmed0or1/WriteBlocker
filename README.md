# WriteBlocker
  A Python script using PyQt6 to create a write-blocker tool, preventing unauthorized file modifications through command-line interface (CMD).

# install & Run
```md
sudo git clone https://github.com/Ahmed0or1/WriteBlocker.git
```
```md
cd WriteBlocker
```
```md
pip install pyqt6
```
```md
python main.py
```
